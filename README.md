# kosynierzy.info

## Installation

Follow the rabbit:

1. git clone `git://github.com/kosynierzy/kosynierzy.info`
2. `cp config/database.yml.example config/database.yml` and fill with proper values
3. `bundle`
4. `bundle exec rake db:setup`
